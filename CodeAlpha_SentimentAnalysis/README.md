# Task 4: Sentiment Analysis

## Project Title
Customer Feedback Sentiment Analysis using TextBlob

## Objective
The objective of this task is to analyze customer feedback and identify whether the sentiment is positive, negative, or neutral using Natural Language Processing techniques.

## Tools Used
- Python
- TextBlob library

## Steps Followed
1. Imported required libraries
2. Defined customer feedback text data
3. Applied TextBlob sentiment analysis
4. Classified sentiment based on polarity score
5. Displayed results and conclusion

## Output
The program analyzes each feedback and displays its sentiment along with polarity score.

## Conclusion
This task helps businesses understand customer opinions and improve their services based on sentiment analysis.
